Advanced Real-time Chat App (Next.js + Tailwind + Supabase)

This workspace is scaffolded to support:
- Supabase Auth (Magic Link / OTP)
- Realtime messaging via `postgres_changes`
- Presence via Supabase Realtime Presence
- Secure Row Level Security (RLS) policies

Environment variables:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
